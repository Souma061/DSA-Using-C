## Bubble Sort

Bubble Sort: -

Input: An array A[1...n] of size n.

Output: The array A[1...n] sorted in non-decreasing order.

Remark: Adjacent elements are swapped repeatedly; after each pass, the largest unsorted element moves to the end.

Steps:

1. Start.
2. For i = 1 to n - 1 do
3. For j = 1 to n - i do
4. If (A[j] > A[j + 1]) then
5. Swap A[j] and A[j + 1].
6. EndIf.
7. EndFor.
8. EndFor.
9. Stop.

## Selection Sort

Selection Sort: -

Input: An array A[1...n] of size n.

Output: The array A[1...n] sorted in non-decreasing order.

Remark: At each step, select the minimum from the unsorted part and place it at the beginning.

Steps:

1. Start.
2. For i = 1 to n - 1 do
3. Set pos = i.
4. For j = i + 1 to n do
5. If (A[j] < A[pos]) then
6. Set pos = j.
7. EndIf.
8. EndFor.
9. Swap A[i] and A[pos].
10. EndFor.
11. Stop.

## Insertion Sort

Insertion Sort: -

Input: An array A[1...n] of size n.

Output: The array A[1...n] sorted in non-decreasing order.

Remark: Build a sorted prefix; insert each new element into its correct position in the prefix.

Steps:

1. Start.
2. For i = 2 to n do
3. Set key = A[i].
4. Set j = i - 1.
5. While (j \ge 1 AND A[j] > key) do
6. Set A[j + 1] = A[j].
7. Set j = j - 1.
8. EndWhile.
9. Set A[j + 1] = key.
10. EndFor.
11. Stop.
