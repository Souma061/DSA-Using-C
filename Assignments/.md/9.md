Reverse Stack using Additional Stack: -

Input: An integer n and a stack S1 containing n elements.

Output: The elements of S1 in reversed order (printed), using an additional stack S2.

Remark: Uses two stacks. By transferring all elements from S1 to S2, the order becomes reversed.

Steps:

1. Start.
2. Read n.
3. For i = 1 to n do
4. Read item.
5. Push item into S1.
6. EndFor.
7. While (S1 is not empty) do
8. x = Pop(S1).
9. Push x into S2.
10. EndWhile.
11. While (S2 is not empty) do
12. Print Pop(S2).
13. EndWhile.
14. Stop.
