Remark: Stack is implemented using an array `STACK[0...MAX-1]` with index `top` (initially -1).

## Push

Push: -

Input: Array STACK[0...MAX-1], integer `top`, and integer `item`.

Output: Updated stack after pushing `item`, or overflow message.

Steps:

1. Start.
2. If (top == MAX - 1) then Print "Stack overflow" and Stop.
3. Set top = top + 1.
4. Set STACK[top] = item.
5. Print "Pushed".
6. Stop.

## Pop

Pop: -

Input: Array STACK and integer `top`.

Output: Prints and removes top element, or underflow message.

Steps:

1. Start.
2. If (top == -1) then Print "Stack underflow" and Stop.
3. Print STACK[top] as popped element.
4. Set top = top - 1.
5. Stop.

## Peek

Peek: -

Input: Array STACK and integer `top`.

Output: Prints top element, or empty message.

Steps:

1. Start.
2. If (top == -1) then Print "Stack is empty" and Stop.
3. Print STACK[top].
4. Stop.

## Display

Display: -

Input: Array STACK and integer `top`.

Output: Prints stack elements from top to bottom.

Steps:

1. Start.
2. If (top == -1) then Print "Stack is empty" and Stop.
3. For i = top down to 0 do
4. Print STACK[i].
5. EndFor.
6. Stop.

## Menu (Program Flow)

Input: User choice and values.

Output: Performs chosen operation repeatedly until exit.

Steps:

1. Start.
2. Repeat
3. Read choice.
4. If choice == 1 then Read item and call Push.
5. ElseIf choice == 2 then call Pop.
6. ElseIf choice == 3 then call Peek.
7. ElseIf choice == 4 then call Display.
8. ElseIf choice == 5 then Stop.
9. Else Print "Invalid choice".
10. EndIf.
11. EndRepeat.
