## Linear Search

Linear Search: -

Input: An array A[1...n] of size n and K is the key to be searched.

Output: A successful message and the location of the array element where K matches, otherwise an unsuccessful message.

Remark: Assume that the array is not empty and the elements are in random order. Array index is 1, 2, ..., n.

Steps:

1. Start.
2. Set i = 1.
3. If (K == A[i]) then
4. Print "Successful at location i".
5. Stop.
6. Else
7. Set i = i + 1.
8. If (i \le n) then
9. Go to Step 3.
10. Else
11. Print "Unsuccessful".
12. Stop.
13. EndIf.
14. EndIf.

## Binary Search

Binary Search: -

Input: A sorted array A[1...n] of size n and K is the key to be searched.

Output: A successful message and the location of the array element where K matches, otherwise an unsuccessful message.

Remark: Array must be sorted (non-decreasing). Array index is 1, 2, ..., n.

Steps:

1. Start.
2. Set low = 1, high = n.
3. While (low \le high) do
4. Set mid = floor((low + high) / 2).
5. If (A[mid] == K) then
6. Print "Successful at location mid".
7. Stop.
8. ElseIf (A[mid] < K) then
9. Set low = mid + 1.
10. Else
11. Set high = mid - 1.
12. EndIf.
13. EndWhile.
14. Print "Unsuccessful".
15. Stop.
