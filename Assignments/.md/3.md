## Length

Length: -

Input: Head pointer `head` of a singly linked list.

Output: Integer `len` = number of nodes.

Steps:

1. Start.
2. Set cnt = 0.
3. Set temp = head.
4. While (temp != NULL) do
5. Set cnt = cnt + 1.
6. Set temp = temp->next.
7. EndWhile.
8. Return cnt.
9. Stop.

## Insert at position pos

Insertion: -

Input: Head pointer `head`, an integer `item`, and a position `pos` (1 to len+1).

Output: Updated linked list after inserting `item` at position `pos` or an error message.

Remark: Position is 1-based. Valid positions are 1, 2, ..., len+1.

Steps:

1. Start.
2. If (pos < 1) then Print "Invalid position" and Stop.
3. Create newNode.
4. Set newNode->data = item.
5. If (pos == 1) then
6. Set newNode->next = head.
7. Set head = newNode.
8. Print "Inserted".
9. Stop.
10. EndIf.
11. Set len = Length(head).
12. If (pos > len + 1) then Print "Invalid position" and Stop.
13. Set temp = head.
14. For i = 1 to pos - 2 do
15. Set temp = temp->next.
16. EndFor.
17. Set newNode->next = temp->next.
18. Set temp->next = newNode.
19. Print "Inserted".
20. Stop.

## Delete at position pos

Deletion: -

Input: Head pointer `head` and a position `pos` (1 to len).

Output: Updated linked list after deleting the node at position `pos` or an error message.

Remark: Position is 1-based.

Steps:

1. Start.
2. If (head == NULL) then Print "List is empty" and Stop.
3. Set len = Length(head).
4. If (pos < 1 OR pos > len) then Print "Invalid position" and Stop.
5. If (pos == 1) then
6. Set temp = head.
7. Set head = head->next.
8. Free temp.
9. Print "Deleted".
10. Stop.
11. EndIf.
12. Set temp = head.
13. For i = 1 to pos - 2 do
14. Set temp = temp->next.
15. EndFor.
16. Set toDelete = temp->next.
17. Set temp->next = toDelete->next.
18. Free toDelete.
19. Print "Deleted".
20. Stop.

## Reverse

Reverse: -

Input: Head pointer `head` of a singly linked list.

Output: Updated list with nodes reversed.

Steps:

1. Start.
2. Set prev = NULL.
3. Set current = head.
4. While (current != NULL) do
5. Set next = current->next.
6. Set current->next = prev.
7. Set prev = current.
8. Set current = next.
9. EndWhile.
10. Set head = prev.
11. Print "List reversed".
12. Stop.

## Search

Search: -

Input: Head pointer `head` and key `item`.

Output: "Found" with position if present, otherwise "Not found".

Remark: Position is 1-based.

Steps:

1. Start.
2. Set temp = head.
3. Set pos = 1.
4. While (temp != NULL) do
5. If (temp->data == item) then Print "Found at position pos" and Stop.
6. Set temp = temp->next.
7. Set pos = pos + 1.
8. EndWhile.
9. Print "Not found".
10. Stop.

## Display

Display: -

Input: Head pointer `head`.

Output: Prints all elements from head to NULL.

Steps:

1. Start.
2. If (head == NULL) then Print "List is empty" and Stop.
3. Set temp = head.
4. While (temp != NULL) do
5. Print temp->data.
6. Set temp = temp->next.
7. EndWhile.
8. Stop.
