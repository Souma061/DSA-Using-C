Remark: Stack is implemented using a singly linked list. `top` points to the top node.

## Push

Push: -

Input: Pointer `top` and integer `item`.

Output: Updated stack after inserting `item` at the top.

Steps:

1. Start.
2. Create newNode.
3. Set newNode->data = item.
4. Set newNode->next = top.
5. Set top = newNode.
6. Print "Pushed".
7. Stop.

## Pop

Pop: -

Input: Pointer `top`.

Output: Deletes the top element and prints it, or prints underflow message.

Steps:

1. Start.
2. If (top == NULL) then Print "Stack underflow" and Stop.
3. Set temp = top.
4. Set top = top->next.
5. Print temp->data as popped element.
6. Free temp.
7. Stop.

## Peek

Peek: -

Input: Pointer `top`.

Output: Prints the top element, or prints empty message.

Steps:

1. Start.
2. If (top == NULL) then Print "Stack is empty" and Stop.
3. Print top->data.
4. Stop.

## Display

Display: -

Input: Pointer `top`.

Output: Prints all stack elements from top to bottom.

Steps:

1. Start.
2. If (top == NULL) then Print "Stack is empty" and Stop.
3. Set temp = top.
4. While (temp != NULL) do
5. Print temp->data.
6. Set temp = temp->next.
7. EndWhile.
8. Stop.

## Menu (Program Flow)

Input: User choice and values.

Output: Performs chosen operation repeatedly until exit.

Steps:

1. Start.
2. Repeat
3. Read choice.
4. If choice == 1 then Read item and call Push.
5. ElseIf choice == 2 then call Pop.
6. ElseIf choice == 3 then call Peek.
7. ElseIf choice == 4 then call Display.
8. ElseIf choice == 5 then Stop.
9. Else Print "Invalid choice".
10. EndIf.
11. EndRepeat.
