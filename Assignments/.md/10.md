Reverse Stack using Additional Queue: -

Input: An integer n and a stack S containing n elements.

Output: Stack S in reversed order.

Remark: Uses an auxiliary queue Q. Move all elements from stack to queue, then from queue back to stack.

Steps:

1. Start.
2. Read n.
3. For i = 1 to n do
4. Read item.
5. Push item into Stack S.
6. EndFor.
7. While (Stack S is not empty) do
8. x = Pop(S).
9. Enqueue x into Q.
10. EndWhile.
11. While (Queue Q is not empty) do
12. x = Dequeue(Q).
13. Push x into Stack S.
14. EndWhile.
15. Print elements of Stack S from top to bottom.
16. Stop.
